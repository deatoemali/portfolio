Project Task

Create any type of media that the team thinks is the best way to tell the story of an existing fantasy narrative script.

Goals

The team decided to create a platformer visual novel game inspired by various indie games alike. The decision to make a game is to increase interactivity in order for the reader to immerse themself into the story.

My Impact

  • Designed, illustrated, and animated assets for a side-scrolling game demo project in a team of 5 for a storytelling assignment, resulting in a final grade of 96%

  • Built a website for the game using Framer, further boosting the immersion and interactivity of the world-building and storytelling of the game

  • Advocated in the whole game design process, including ideation, project planning, group meetings, and level design

Key Takeaways

  • Successfully learned something out of my comfort zone which was illustrating, animating 2D assets, and website making.

  • Learned the process of game developing, starting from deisgning the mechanics and aesthetics, to creating the assets and level mapping of the game.
